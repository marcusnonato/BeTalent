# BeTalent - Teste Prático Mobile

Olá!

Bem-vindo(a) ao meu teste! :)

Este aplicativo exibe uma tabela de colaboradores, consumindo dados de uma API simulada via json-server. 

A interface permite pesquisar por nome, cargo e telefone, e realiza a formatação de datas e telefones conforme solicitado.

#

# Comandos

Iniciar servidor json-server: npm run server

Iniciar proejto react native no android: npx react-native run-android

Script criado para rodar o servidor json

"server": "json-server --watch app/db/database.json"

obs: não esqueça de mudar o baseURL para seu locahost! :)

está no arquivo app/lib/api.ts
